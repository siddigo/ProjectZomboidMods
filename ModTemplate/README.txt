Utilize esse template para criar um novo mod.

Adicionar o banner do mod para o jogo no formato png (mesmo nome utilizado em mod.info)

Esta pasta refere-se apenas ao mod em si. Para adicionar dentro da Steam, o mod precisa estar dentro da seguinte estrutura:

WorkShopName>Content>mods>

Dentro da pasta WorkShopName ainda é necessário criar:
- Banner preview.png (256x256)
- Documento workshop.txt contendo:
version=1
id=IdSteam, em branco para novo envio.
title=Mod Name
description=Mod Description.
tags=Build 41
visibility=private
